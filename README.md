Background
This project is to analyze a dataset with 16,598 rows of game sales data. The IPython file performs data pre-processing, data cleasing, and data visualization.

Problems that were analyzed
1. what types of games are more popular?
2. what are the popular conditions of different game genre?
3. which popular games are sold the most on different platforms?
4. which game companies are making the most profits?
5. Are there observed trends in the game genre that consumers liked?
6. what are the features of the most popular games?
